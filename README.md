Firebase Job Portal (firebasedddd)This is a simple Android application for a job portal built with Kotlin and Firebase. The app allows users to register, log in, post new job opportunities, and view the jobs they have created.Key Features•Firebase Authentication: Secure user registration and login using email/password and Google Sign-In.•Cloud Firestore: A real-time NoSQL database for storing job and user data.•Post Jobs: Authenticated users can post new job listings.•View All Jobs: A main feed that displays all available jobs from all users.•View My Jobs: A dedicated screen showing only the jobs posted by the logged-in user.•App Check Security: Protects backend resources from abuse using Firebase App Check.Recent Updates & Fixes•Login and Account Creation Bug Fixed: Resolved critical bugs in the authentication flow, ensuring a stable and reliable user login and registration experience.•Apply Button Added: The UI for job listings now includes an "Apply" button, setting the stage for future applicant management features.•Tab with My Jobs Was Created: A new "My Jobs" screen has been implemented, allowing users to easily view and manage the job listings they have personally created.
